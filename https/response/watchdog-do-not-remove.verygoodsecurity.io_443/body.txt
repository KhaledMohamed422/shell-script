  -H "Content-type: application/json" \
  -d '{"aaaaa": "bbbb"}' | jq .

  "url": "http://boomerang.verygoodsecurity.io/post",
  "headers": {

<h4>Note</h4>
<p>
    <ul>
        <li>response is always json with content type "application/json"</li>
        <li>if you pass request with ContentType "application/json" - then payload will be parsed and placed into json field</li>
        <li>only 3 types of requests are supported
            <ul>
                <li>GET - /get</li>
                <li>POST - /post</li>
                <li>PUT - /put</li>
            </ul>
        </li>
    </ul>
