    <h1>Super 6 API</h1>

    <p>The Super 6 API provides access to Super 6 data, and allows authorised clients to interact with the game. The API follows a RESTful design, and provides JSON responses.</p>

    <p>The API allows consumers to get information about the current state of Super 6, such as the round status, prizes available and number of entries. Other details about the game are available, such as the leaderboard standings and user's results. Please see the technical documentation for a complete list of endpoints and data available.</p>
